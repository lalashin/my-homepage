# 신인숙 | 개인 홈페이지

프론트엔드 개발자 신인숙의 개인 홈페이지입니다.

## 소개

웹 접근성과 성능을 함께 챙기며, 명확한 코드와 일관된 디자인을 지향하는 개인 포트폴리오 사이트입니다.

## 배포

- **라이브 사이트**: [https://my-homepage-three-gamma.vercel.app/](https://my-homepage-three-gamma.vercel.app/)

## 구성

| 섹션 | 내용 |
|------|------|
| 소개 | 자기소개 및 개발 철학 |
| 관심사 | UI/UX, 웹 표준, 학습 |
| 연락처 | 이메일 및 GitHub 링크 |

## 기술 스택

- HTML5 (시맨틱 마크업, 웹 접근성)
- CSS3 (CSS Variables, Grid, Flexbox, 반응형 디자인)
- Vanilla JavaScript
- Google Fonts (Plus Jakarta Sans)

## 주요 특징

- **반응형 디자인** — 모바일·태블릿·데스크탑 대응
- **웹 접근성** — 스킵 내비게이션, aria 속성, 시맨틱 HTML
- **모바일 햄버거 메뉴** — 640px 이하에서 드롭다운 네비게이션
- **부드러운 인터랙션** — 호버 애니메이션, 스크롤 이동

## 파일 구조

```
my-homepage/
├── index.html    # 메인 페이지
├── index1.html   # 추가 페이지
├── style.css     # 스타일시트
└── script.js     # 네비게이션 토글 및 연도 자동 갱신
```

## 로컬 실행

별도 빌드 없이 `index.html`을 브라우저에서 바로 열어 확인할 수 있습니다.
